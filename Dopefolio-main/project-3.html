Project Overview

Implemented a web scraper which scrapes research data from google scholar and returns citation data scraped in the form of an excel sheet. This can be used to compare the total research output of a particular department of a university or the individual research output of a professor.

Also implemented an automatic login script for codetantra online classes. The script will automatically login to all the online classes provided the machine is switched on and has internet.
